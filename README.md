# Quantum-simulator-
this is a quantum simulator project, that demonstrates the X gate, Hadamard gate and the CNOT gates. I have also made a bell state simulator with 2 qubits.
how to run:
1. have python installed
2. install required packages/dependencies
3. run it using : python bellstate.py
